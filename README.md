# 🎬 Movie Explorer Dashboard

A modern movie discovery dashboard built with **React, Vite, Redux Toolkit, and Tailwind CSS**.

## 🚀 Features
- Search movies in real-time
- Filter by genres
- Pagination support
- Favorites management
- Responsive UI

## 🛠 Tech Stack
- React + Vite
- Redux Toolkit
- Tailwind CSS
- TMDB API

## ▶️ Run Locally
```bash
npm install
npm run dev
